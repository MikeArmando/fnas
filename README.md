# Five Nights At School

## Objetivo del juego:

Resolver operaciones matemáticas básicas mientras el jugador se divierte participando en el juego. Esto con el propósito de fortalecer las habilidades matemáticas básicas del jugador mientras interactúa con un juego de terror interactivo y estimulador, que sea para entretener y enseñar al mismo tiempo.

## Edades objetivo:

Para 3ro de primaria (8-9 años)

## Características principales del juego:

Es un juego amigable de terror para niños, maneja elementos de terror sin contener ningún material “extremo”.
juego c/c++ librería raylib
de un solo nivel
sistema de puntajes
adaptable a cualquier problema matemático básico
Las tareas están diseñadas para tomar en cuenta las necesidades de aprendizaje del niño, dando suficiente tiempo e intentos para permitir el aprendizaje de forma sencilla.
Habrá una sección principal en donde se explicarán los ejercicios a realizar.
3 secciones, en donde a cada sección le corresponderá alguna operación matemática, sumas, restas y multiplicaciones.

### Navegar en los archivos del programa

Buscar dentro del archivo actual: Ctrl+F (Windows/Linux)

Buscar en todo el proyecto (files): Ctrl+Shift+F

En files to include/exclude puedes poner src/\*_, _.js, o !node_modules para filtrar.

Navegar por símbolos (mejor para funciones/variables)

Símbolos en el archivo (funciones, clases, variables): Ctrl+Shift+O

Símbolos en todo el workspace (ir a función/variable por nombre): Ctrl+T

Alternativa: abre Quick Open (Ctrl+P / Cmd+P) y escribe @ para símbolos del archivo, # para buscar en contenido del archivo.
